List

This is how you make an unorderd list:

  • Add a "ul" element (stands for unorderd list).
  • Add each item on its own "li" element.
  • They don't need to be in particular order.

This is how the ordered list look like:

  1. Using the "ol" element instead of "ul".
  2. Notice the new "ol" element wrapping everyting.
  3. But the list items "li" are the same.
  4. Also note the number increment on their own.
  5. You should be noticing things in this precise order, because this is an ordered list.
